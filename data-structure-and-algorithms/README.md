## Setup Java Environment \(macOS\)

1. Install Java Development Kit

2. Create "HelloWorld.java"

```java
public class HelloWorld { 
    public static void main(String[] args) { 
    System.out.println("Hello, World"); 
    }
}
```

3. Compile

```text
javac HelloWorld.java
```

4. Run

```text
java HelloWorld
```
